# PolicyHQ Design Shotgun

Use this before major UI changes.

## Output Format

Produce up to three directions:

- Conservative: preserves current layout and improves clarity.
- Figma-exact: follows the current Figma/mockup most closely.
- Operational SaaS: optimizes density, scanning, and daily workflows.

For each direction, include:

- Layout idea
- Best use case
- Tradeoff
- Mobile impact

End with one recommendation. Do not change code until the user approves one direction.
